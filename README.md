# Institutional Attendance Management System

## 📌 Overview
The Institutional Attendance Management System is a cloud-based application developed using Zoho Creator, a low-code development platform. This system digitizes and automates the process of recording, managing, and analyzing student attendance within an academic institution.

The application allows faculty to record attendance for each class session, manage student records, generate monthly reports, and view attendance analytics through an interactive dashboard. It eliminates manual attendance registers and provides real-time access to attendance information.

---

## 🎯 Objective
The primary goal of this system is to:

- Eliminate manual attendance recording using paper registers
- Provide a centralized platform to manage student attendance data
- Automatically calculate attendance percentages
- Generate monthly attendance reports
- Provide analytics dashboards for monitoring attendance performance
- Improve accuracy, efficiency, and transparency

---

## 🚀 Key Features

### 👨‍🎓 Student Management Module
- Add and manage student details
- Store department, semester, and contact information
- Maintain enrollment and student status records

### 🧑‍🏫 Attendance Recording Module
- Record attendance for each student
- Select course, class session, and date
- Mark attendance status (Present, Absent, Late, Excused)
- Add remarks if required

### 📊 Attendance Analytics Module
- View attendance percentage for each student
- Track total present and total conducted classes
- Identify students with low attendance
- Monitor attendance trends

### 📅 Monthly Reports Module
- Generate monthly attendance reports
- View attendance summary by student and course
- Store report data and download links
- Track report generation details

### 📈 Dashboard Module
- View total present count
- View total classes conducted
- View average attendance percentage
- Visual charts for attendance analysis
- Quick access to attendance forms
